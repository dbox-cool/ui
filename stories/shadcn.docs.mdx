# Import [shadcn](https://ui.shadcn.com/) Elements

Follow [this guide](https://ui.shadcn.com/docs/installation/manual). **THERE'S NO NEED to follow step 3**, you can use any icon pack you want.

Here's a js+jsdocs version of the `cn` function:

```js
import { clsx } from "clsx";
import { twMerge } from "tailwind-merge";

/**
 * A way to safely merge classnames
 * @param {import("clsx").ClassValue[]} inputs Array of classes to merge
 * @returns {string} String of all classes merged 
 */
export function cn(...inputs) {
  return twMerge(clsx(inputs));
}
```
